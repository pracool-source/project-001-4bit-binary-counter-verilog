# Project 001: 4-Bit Binary Counter using Verilog HDL

## Overview

A 4-bit binary counter designed and implemented using Verilog HDL.

The counter increments its value on every positive clock edge and resets to zero when the reset signal is asserted.

## Features

- 4-bit binary counting
- Sequential logic design
- Verilog HDL implementation
- Reset functionality
- Simulation-ready architecture
